# [OAuth 2.0 - Proteja suas aplicações com o Spring Security OAuth2](https://www.casadocodigo.com.br/products/livro-oauth)

Código desenvolvido para entendimento do protocolo (ou melhor falando framework, vide [RFC](https://tools.ietf.org/html/rfc6749)) OAuth 2.0 com auxílio do Spring Security.

O projeto original está no [repositório](https://github.com/adolfoweloy/livro-spring-oauth2) que o Adolfo criou e disponibilizou no livro. Tive que alterá-lo para corresponder a versão atual do Spring e também para não deixá-lo dependente do MySQL, portanto passei para um banco em memória (no caso H2) para facilitar a execução no projeto.
